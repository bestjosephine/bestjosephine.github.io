### 步骤

1. **开通 DashScope**
   - 打开 https://bailian.console.aliyun.com/
   - 用阿里云账号登录（没有的话先注册，支持支付宝/钉钉登录）
   - 首次使用需要开通"百炼"服务，按提示操作即可

2. **创建 API Key**
   - 开通后，进入 https://bailian.console.aliyun.com/?apiKey=1#/api-key
   - 点击"创建新的 API-KEY"
   - 复制生成的 Key（格式类似 `sk-xxxxxxxxxxxxxxxxxxxxxxxx`）
   - ⚠️ 只在创建时显示一次，记得保存好

3. **免费额度**
   - 通义千问有免费额度，新用户每月有一定量的免费调用次数
   - 具体额度见：https://help.aliyun.com/zh/model-studio/developer-reference/tongyi-qianwen

4. **可用的免费模型**
   - `qwen-turbo`：速度快，适合日常使用（免费额度高）
   - `qwen-plus`：能力强一些，也有免费额度
   - `qwen-max`：最强模型，免费额度较少
